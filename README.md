# Calculator Application

This is a simple calculator application built using HTML, CSS, and JavaScript. It provides a user-friendly interface for performing basic arithmetic operations.

## Project Structure

The project consists of the following files:

- **index.html**: Contains the structure of the calculator application, including buttons for digits and operations. It links to the CSS for styling and the JavaScript for functionality.
  
- **style.css**: Contains the styles for the calculator application, defining the appearance, layout, colors, fonts, and button styles.
  
- **script.js**: Contains the JavaScript code that handles button clicks, updates the display, performs calculations, and manages the calculator's state.

## Usage

1. Open `index.html` in a web browser to view the calculator.
2. Click on the buttons to perform calculations.
3. Use the "AC" button to clear the display and reset the calculator.
4. Use the "DEL" button to delete the last character in the display.

## Setup

No additional setup is required. Simply open the `index.html` file in your preferred web browser to start using the calculator.
